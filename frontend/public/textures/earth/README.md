# Earth Textures

Drop high-quality Earth maps in this directory to enable realistic rendering in the mission viewport.

Expected filenames:

- `earth_day_4k.jpg`
- `earth_night_4k.jpg`
- `earth_clouds_4k.png`
- `earth_normal_4k.jpg`
- `earth_specular_4k.jpg`

If these files are not present, the frontend uses procedural fallback textures.
