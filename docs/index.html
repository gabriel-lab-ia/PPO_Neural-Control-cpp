Neuro Motor CPP

A formal C++ reinforcement learning project centered on PPO, MuJoCo simulation, metric exports, and a browser-based 3D neural-network viewer.

This site is the static publishing surface for the repository demo. It provides direct access to the trained-policy visualization, benchmark summary, and exported artifacts.

Project Focus

PPO is implemented in modern C++ using LibTorch, with a clean environment abstraction that supports a default baseline and optional MuJoCo backends.

The viewer exposes policy structure, connection weights, and sampled live activations in a standalone HTML artifact suitable for static hosting, now with touch-friendly controls for mobile devices.

Training Core

Rollout collection, generalized advantage estimation, PPO clipping, entropy preservation, and critic stabilization are implemented in C++ and exported through reproducible metric logs.

MuJoCo Integration

The repository includes a MuJoCo cart-pole adapter and local XML assets, making it straightforward to extend toward richer control benchmarks.

Web Visualization

The demo artifacts are static and browser-friendly, with direct desktop and mobile interaction, which makes GitHub Pages a practical zero-backend publishing path.
