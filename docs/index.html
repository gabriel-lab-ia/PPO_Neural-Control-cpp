Orbital Neural Control CPP

CPU-first reinforcement learning systems baseline for orbital control research and engineering. The active CLI runtime provides reproducible train, eval, and benchmark flows with structured artifacts and SQLite persistence.

Optional tracks include MLflow/ONNX MLOps, telemetry backend streaming, and a Next.js mission dashboard. These are expansion modules, not hidden baseline requirements.
